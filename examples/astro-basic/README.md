# Astro Basic Example

Tiny Astro-shaped fixture for VibeRouter routing examples.

Useful task:

```sh
viberouter scope "Add FAQ section to homepage" --root examples/astro-basic
```

Expected behavior:

- project type: `astro`
- tier: `cheap`
- allowed files include `src/components/sections/FAQ.astro`, `src/content/site.json`, and `src/pages/index.astro`
